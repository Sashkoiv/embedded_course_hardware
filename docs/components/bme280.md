# BME280 – Temperature, Humidity, Pressure Sensor

## Overview

The BME280 is a digital environmental sensor measuring:

- Temperature
- Humidity
- Atmospheric pressure

In this course it is used to practice:
- I2C communication
- SPI communication
- Sensor configuration
- Reading and converting digital data

## Key Specifications

- Supply voltage: **1.7V – 3.6V**
- Logic level: **3.3V**
- Interface: **I2C / SPI**
- I2C address: **0x76 or 0x77**
- Low power consumption

⚠ Sensor core is 3.3V only. Verify your breakout board.

---

## Pinout (Typical Breakout)

![BME280 module](../img/bme280/pinout.png)

| Pin | I2C Function | SPI Function | Notes |
|------|--------------|--------------|--------|
| VCC | Power | Power | 3.3V |
| GND | Ground | Ground | Common ground required |
| SDA | I2C Data | MOSI | Master → Sensor |
| SCL | I2C Clock | SCK | Serial clock |
| CS  | — | Chip Select | Active LOW |
| SDO | Address select | MISO | LOW=0x76, HIGH=0x77 |

⚠ Use pull-up resistors (2k2 - 4.7k - 10kΩ) if not present on module.

---

## Electrical Notes

- Do not apply 5V directly to sensor core.
- Always use common ground.
- Address depends on SDO pin level.

---

## Common Student Mistakes

- Wrong I2C address
- Missing pull-ups
- Swapped SDA/SCL
- Using 5V logic without checking module
